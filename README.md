# Skill Test BE - Cinema Ticket API
## Instalasi
1. Clone repository
``` bash
git clone https://github.com/fahrikurniawan99/mcm-be.git
```
2. Install dependencies
``` bash
npm install 
```
4. Create a MySQL database named cinema_db
6. Run Development Mode
``` bash
npm run dev
```
6. Run Production Mode
``` bash
npm run start
```


## API Endpoints
- `/api/films`
- `/api/studios`
- `/api/showtimes`
- `/api/tickets`
